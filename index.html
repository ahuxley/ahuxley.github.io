CardMap Inventory Scraper

This Python scraper is the backend ingestion service behind CardMap. It collects multi-store inventory, standardizes listings, and prepares searchable records for the app.

Unlike the main CardMap product work, this project is focused on source compatibility, normalization rules, concurrency, and run reliability across multiple retailer ecosystems.

  • Integrated BinderPOS/Shopify and TCGplayer Pro into one normalized ingestion pipeline
